Flower Website Description

This Flower Website is created using HTML, CSS, and JavaScript.
HTML is used to build the structure of the website, such as the navigation bar,
flower product sections, and cart page. CSS is used to design the layout,
add colors, fonts, spacing, and make the website visually attractive.
Google Fonts were also included to improve the typography of the website.
JavaScript is used to implement the Add to Cart functionality and store
cart items using localStorage. This project helped me understand
basic web development concepts like page structure, styling, and simple
interactive features.
